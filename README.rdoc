== upload-s3-demo app

This is a sample website application built using Ruby on Rails with the basic functionality of create a POST with attached IMAGES and FILES hosted in Amazon S3. The app is deployed in Heroku.

{Demo}[https://salty-river-37135.herokuapp.com]

* Ruby 2.0 above, Rails 4.2.1; See more: {Gemfile}[https://github.com/erikathea/upload-s3-demo/blob/master/Gemfile]
* Local setup: 
  Update config files:
    database.yml
    aws.yml
